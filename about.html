Welcome to Cardirectory.online!

What is Cardirectory?

Cardirectory is a website that gathers all kinds of documents about cardistry.
I came up with the name when I first started planning the site. I wanted something that clearly described its purpose, and that’s when “directory” and “cardistry” clicked. Just like “cardistry” (card + artistry), I created “Cardirectory” (card + directory).

What can I find in here?

You can find some cardistry brands, markets, community, etc.

Why you made this website?

Before I started building this website, I often struggled to find the websites or social media accounts of cardistry brands and shops. I had to search on Google or look through Instagram every time, which was really inconvenient.And if I couldn’t remember the name, it was almost impossible to find them again. So I wanted to create a website that gathers all brands, markets, and more in one place. That’s how Cardirectory started.

I’m not very good at coding, so I built and updated this site with the help of AI (vibe coding). There might be some bugs, but I’ll fix them as I go, so thanks for your understanding! I’m not sure how long this site will last, but I hope you enjoy it while it’s here(?). If you have any suggestions, feel free to DM me below. Thank you!
